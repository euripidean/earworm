About Earworm

Earworm is a music review site that allows users to search for albums and artists, and read reviews of albums. Users can also create their own reviews of albums.

This site was developed by euripidean for a backend web development class as part of the Applied Computer Science course at Dominican University of California.

Future Development Ideas

  • API integration with Spotify, allowing a user's Earworm account to be their Spotify account.
  • Ingration with ticket sellers for artist tours
  • Create artists/albums via integration rather than manual input
  • Build on a community aspect of listening to music
